FONTLOG for the Blinker fonts

This file provides detailed information on the Blinker Font Software.
This information should be distributed along with the Blinker fonts
and any derivative works.


Basic Font Information

Blinker is a Unicode typeface family with a Latin Extended glyph set, supporting multiple languages that use the Latin script. It could be expanded to support other scripts.
